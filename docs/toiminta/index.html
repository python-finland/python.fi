Yleistä

Tiedotuskanavat, keskusteluryhmät, yms:

  • PIG-Fi-keskusteluryhmä (Google Groups)
  • @PythonFinland Twitterissä
  • Facebook-ryhmä
  • IRC-kanava #python.fi (IRCNet)
  • PyCon Finland -konferenssi

Yleistä Python-aiheista keskustelua käydään PIG-Fi-keskusteluryhmässä, jonne myös kaikki Python Suomi ry:n tiedotteet kaiutetaan. Reaaliaikainen keskustelu hoituu #python.fi-kanavalla IRCnetissä.

Suurin tapahtumamme on vuosittainen PyCon Finland -konferenssi. Järjestämme lisäksi paikallisia illanistujaisia ja käymme puhumassa oppilaitoksissa. Osallistumme muiden järjestämiin tapahtumiin, joissa promotoimme Pythonia ja siihen liittyviä teknologioita.
